# assets/images

Esta carpeta está destinada a almacenar todos los recursos de imágenes del proyecto.

## Estructura Recomendada

```
images/
├── icons/          # Iconos de la aplicación
│   ├── favicon.ico
│   ├── logo.png
│   └── logo.svg
├── backgrounds/    # Imágenes de fondo
├── screenshots/    # Capturas de pantalla para documentación
└── social/         # Imágenes para redes sociales (Open Graph, etc.)
```

## Formatos Recomendados

- **PNG**: Para iconos y gráficos con transparencia
- **SVG**: Para gráficos vectoriales escalables
- **WEBP**: Para imágenes optimizadas (con fallback)
- **JPG**: Para fotografías

## Nomenclatura

- Usa kebab-case: `mi-imagen.png`
- Nombres descriptivos: `logo-principal.svg`
- Incluye dimensiones si es relevante: `banner-1200x600.jpg`

## Optimización

Antes de agregar imágenes:
1. Comprime las imágenes (TinyPNG, Squoosh, etc.)
2. Usa dimensiones apropiadas (no subas imágenes más grandes de lo necesario)
3. Considera formatos modernos (WebP, AVIF)
